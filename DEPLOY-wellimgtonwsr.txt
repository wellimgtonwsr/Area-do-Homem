FAQ - PROBLEMAS COMUNS
======================

❌ "git: command not found"
→ Git não está instalado
→ Baixe em: https://git-scm.com/download/win

❌ "node: command not found"
→ Node.js não está instalado
→ Baixe em: https://nodejs.org/

❌ "fatal: destination path 'existing' already exists"
→ Já existe um repositório
→ Cole: git config user.name "wellimgtonwsr"

❌ "fatal: Authentication failed"
→ Senha do GitHub errada ou expirou
→ Vá em: https://github.com/settings/tokens
→ Crie um novo token
→ Cole como senha

❌ Railway mostra erro "Build failed"
→ Vá: Railway Dashboard → seu projeto → Logs
→ Procure por error ou failed
→ Se tiver "cannot find module", rode npm install novamente

❌ Imagens não carregam
→ Via de regra é problema de cache
→ Atualize o navegador (Ctrl+F5)
