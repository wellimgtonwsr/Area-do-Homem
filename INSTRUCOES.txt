## 🛫 PASSO 3: Deploy no Railway

### 3.1 Acesse Railway
- Vá para: https://railway.app/dashboard
- Faça login (ou crie conta se não tiver)

### 3.2 Crie Novo Projeto
1. Clique **"Create New"**
2. Clique **"Project"**
3. Selecione **"Deploy from GitHub"**
4. **Autorize** o Railway acessar seu GitHub
5. Procure por **`area-do-homem`**
6. Selecione o repositório
7. Clique **"Deploy"**

### 3.3 Espere o Deploy
- Railway vai:
  1. Fazer clone do repositório
  2. Instalar dependências (`npm install`)
  3. Iniciar o servidor
  4. Gerar uma URL pública

**Tempo:** ~2-3 minutos

### 3.4 Acesse seu Site
- Vá até: https://railway.app/dashboard
- Seu projeto estará listado
- Clique nele
- Você verá a URL como: `https://seu-projeto-xxxxx.up.railway.app`

## 🆘 Precisa de Ajuda?

### Erro no Git Push?
```bash
git config --global user.name "Seu Nome"
git config --global user.email "seu@email.com"
git push -u origin main
```

### Erro no GitHub?
- Verifique se você está logado
- Repository name deve ser `area-do-homem`

### Railway não inicia?
- Vá até: Railway Dashboard → seu projeto → **Logs**
- Procure por erros
- Verifique se `package.json` existe

### Porta fechada?
- Railway abre automaticamente
- Não precisa fazer nada
